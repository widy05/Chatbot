# 🤖 Nova — AI Chatbot for E-Commerce & Document Analysis

Intelligent chatbot powered by OpenAI GPT, built with Python and Flask.
Designed to handle customer support, product questions, order tracking
and PDF document analysis for businesses.

## ✨ Features
- Natural language understanding via OpenAI API
- PDF document reading and analysis
- Custom system prompt per business
- Persistent conversation history
- Clean and responsive web interface
- Easy to deploy and integrate

## 🛠️ Tech Stack
- Python / Flask
- OpenAI API (GPT)
- HTML / CSS / JavaScript

## 💡 Use Cases
- E-commerce customer support
- Product catalog assistant
- Document Q&A (PDF analysis)
- Business FAQ automation

## ▶️ Usage
```bash
pip install flask openai
python app.py
```
